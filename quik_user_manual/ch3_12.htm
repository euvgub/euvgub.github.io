    Назначение

    Просмотр информации о запретах на операции клиентов.

    Формат таблицы

    Каждая строка таблицы описывает отдельный запрет на операции клиента. В столбцах таблицы отображаются следующие параметры:

    Параметр Значение
    Фирма Идентификатор фирмы
    Тип контракта Тип контракта. Возможные значения:
    • «Фьючерсы»;
    • «Опционы»
    Правило № Номер правила запрета
    Тип правила Тип запрета. Возможные значения:
    • «Глобальное» – запрет распространяется на все инструменты;
    • «На инструмент» – запрет распространяется на определенный инструмент
    Код инструмента Код инструмента, на который распространяется запрет. Для типа правила «Глобальное» поле не заполняется
    Код базового актива Заполняется для правил по базовым активам
    Секция Тип секции торговой площадки. Возможные значения:
    • «Фондовая»;
    • «Денежная»;
    • «Товарная»
    Счет Код торгового счета
    Приоритет Приоритет правила. Возможные значения (в порядке убывания приоритета):
    • «Клиентский код, инструмент»;
    • «Клиентский код»;
    • «Клиентский код, все БА»;
    • «Код БФ, инструмент»;
    • «Код БФ, БА»;
    • «Код БФ, все БА»;
    • «Код РФ, инструмент»;
    • «Код РФ, БА»;
    • «Код РФ, все БА»
    Тип состояния Возможные значения:
    • «Всё разрешено»;
    • «Запрет открытия позиций»;
    • «Запрет всех торговых операций»;
    • «Запрет открытия позиций в продажу»;
    • «Запрет брокера на подачу заявок на экспирацию»;
    • «Заявки на экспирацию разрешено подавать только Главному трейдеру»
    Состояние исполнения Статус правила. Возможные значения:
    • «Не разрешено»;
    • «Разрешено»
    Инициатор Инициатор запрета. Возможные значения:
    • «БФ»;
    • «Главный трейдер РФ»;
    • «Администратор КЦ»;
    • «Администратор ТС»
    Тип инициатора Тип инициатора запрета. Возможные значения:
    • «Запрет был выставлен Администратором торгов или клиринга и не может быть изменён участником торгов»;
    • «Запрет был выставлен участником торгов и может быть изменён»
