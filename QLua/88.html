Индикаторы технического анализа
7.1
Общие сведения
Индикаторы
технического анализа
представляют
собой отдельный класс скриптов, которые
удовлетворяют определенным условиям и расположены в папке
LuaIndicators
в каталоге терминала.
Если папка отсутствует в каталоге,
необходимо
создать ее вручную. Список скриптов недоступен из
диалога
Сервисы
/ LUA
скрипты…
.
При добавлении нового индикатора на график плагин
qlua
сканирует папку
LuaIndicators
, проверяет
файлы с расширением lua и luac (скомпилированные скрипты lua) на соответствие следующим
требованиям:
•
определена функция
Init;
•
определена функция OnCalculate;
•
определена таблица Lua с именем Settings, в которой есть поле «Name».
Пример минимального корректного кода для индикатора:
Settings={}
Settings.Name = "minimal"
function Init()
return 1
end
function OnCalculate(index)
return 1
end
Список доступных индикаторов передается
в модуль
qchart
и в дальнейшем доступен из
стандартного
диалога добавления индикатора на графике:
