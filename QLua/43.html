AddColumn
Функция добавляет колонки в таблицу с идентификатором «t_id».
Формат
вызова

NUMBER

NUMBER

NUMBER

Параметры:
•

код параметра,

•

–
название колонки;
•
is_default
–
параметр не используется;

тип данных в колонке, одна из следующих констант:

3.15.2
AllocTable
Функция создает структуру
,
описывающую
таблицу.
Формат вызова:
NUMBER
AllocTable()
Функция возвращает целочисленный идентификатор таблицы, предназначенный для совершения с ней
дальнейших операций.
