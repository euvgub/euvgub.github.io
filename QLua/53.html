Функции для заказа параметров Таблицы
текущих торгов
3.18.1
ParamRequest
Функция заказывает получение параметров Таблицы текущих торгов.
BOOLEAN
ParamRequest
(STRING
class_code,
STRING
sec_code,
STRING
db_name
)
Для корректной работы функции включите
в настройках Рабочего места
QUIK
признак получения данных «Исходя из
настроек открытых пользователем
таблиц» (меню Система / Настройки / Основные
настройки..., раздел
«Программа» / «Получение данных»)
.
Параметры:
•
class_code
–
код класса;
•
sec_code
–
код инструмента
;
•
db_name
–
код параметра.
В случае успешного завершения функция возвращает «
true
», иначе
–
«
false
».
