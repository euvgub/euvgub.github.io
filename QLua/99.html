8.
Потокобезопасные функции для
работы с таблицами
Lua
Одновременная работа с таблицами из функций обратного вызова скрипта и функции main() может
приводить к неопределенным ситуациям.
Для решения этой проблемы qlua.dll предоставляет
потокобезопасные аналоги стандартных функций
Lua.
Выполнение потокобезопасной
функции
блокирует выполнение кода в другом
потоке до
окончания работы функции.
