    Формат таблицы

    Каждая строка таблицы описывает отдельный тип перевода. В столбцах таблицы отображаются следующие параметры:

    Параметр Значение
    Класс Код класса
    Фирма Идентификатор фирмы
    Тип Идентификатор типа перевода
    Описание Дополнительная информация

    Доступные функции

    Данные из таблицы доступны для копирования, вывода через DDE-сервер и экспорта по ODBC.

    Функции, доступные для данной таблицы, могут быть вызваны из пункта меню Действия или контекстного меню таблицы.
