В папке с РМ
QUIK
создадим папку Include, куда и сохраним файл ma.lua.
Код индикатора с использованием сохранённого файла принимает следующий вид:
dofile(getWorkingFolder() .. "\\Include\\ma.lua")
Settings =
{
Name = "EMA",
period = 50,
value_type = "C",
line=
{
{
Name = "1",
Color = RGB(255, 0, 0),
Type = TYPE_LINE,
Width = 2
}
}
}
function Init()
myEMA = cached_EMA()
return 1
end
function OnCalculate(index)
return myEMA(index, Settings.period, Settings.value_type)
end
Функция getWorkingFolder возвращает путь папки с файлом info.exe. В примере последний
параметр не используется и по умолчанию в функции инициализируется значением
2/(Setting.period+1).
