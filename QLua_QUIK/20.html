3.6
Доступ к данным
Каждый индикатор привязан к источнику данных. Для доступа к данным скрипт использует
следующие функции:
•
O(i);
•
H(i);
•
L(i);
•
C(i);
•
V(i);
•
T(i).
За исключением функции T(), все они возвращают соответствующее значение для указанного
бара –
Open, High, Low,
Close и Volume. Функция T() возвращает таблицу, которая содержит
время указанного бара.

функцию:

}
}
}
